# AGENTS.md

## Environment
- Use `npm` for all tasks, including installing dependencies and running scripts.

## Code Style
- Follow ESLint rules.

## Dependencies
- Keep dependencies minimal.
- Discuss before adding new major packages.

## Quality Assurance
- Before committing, run the `build` and `lint` commands.
- Ensure there are no compilation errors or lint warnings.

## Committing
- Write commit messages following Conventional Commits with Gitmoji.
  - Use the format: "<emoji> <type>[optional scope]: <description>".
- Write the description in bullet points and summarise the changes.
